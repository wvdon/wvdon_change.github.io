##### 如何去学习

> 在学习过程中我一直在想，如果我是一个小孩子，你要怎么样才能对我讲明白这些知识点，
>
> 我不停的反问自己  
>
> ```
> 它是什么？
> ```
>
> ```
> 它有什么用？
> ```
>
> ```
> 如何实现它？
> ```
>
> ```
> 还有没有其他的方法呢？
> ```
>
> 人只有在不停的去审视自己的想法的时候才能走的更快，走的更稳。
